# Browse Markets

Learn how to discover and explore prediction markets on AchMarket.

## Overview

The Home page displays all prediction markets with powerful filtering and sorting options.

## Interface Guide

### Market Cards

Each market appears as a card showing:

```
┌─────────────────────────────────────────────────────────┐
│ [Image]  Sports                    Active              │
│                                                         │
│ Will Manchester United win the FA Cup 2024?             │
│                                                         │
│ Yes: 65%  ───────────────────────────────────  35%: No  │
│                                                         │
│ Ends in: 3 days          Volume: $12,500 USDC          │
└─────────────────────────────────────────────────────────┘
```

| Field | Description |
|-------|-------------|
| Image | Market header image |
| Category | Market category badge |
| Stage | Active/Resolved/Cancelled |
| Title | Market question |
| Probability | Current outcome odds |
| End Date | When market resolves |
| Volume | Total USDC traded |

## Filtering Markets

### By Category

Filter by topic:

| Category | Description |
|----------|-------------|
| Crypto | Cryptocurrency events |
| Sports | Sports outcomes |
| Politics | Political events |
| Entertainment | Movies, awards, etc. |
| Science | Scientific discoveries |
| Other | Miscellaneous |

### By Stage

Filter by market status:

| Stage | Description |
|-------|-------------|
| Active | Trading ongoing |
| Resolved | Market decided |
| Cancelled | Market cancelled |
| Expired | Not resolved in time |

### Search

Search markets by title:
- Type in search box
- Results update in real-time
- Case insensitive

## Sorting Options

### Sort By

| Option | Description |
|--------|-------------|
| Newest | Recently created |
| Ending Soon | Deadline approaching |
| Highest Volume | Most traded |
| Most Participants | Most unique traders |

### Default Sort

Default is **Newest**, showing fresh markets first.

## Market Information

### Probability Display

The probability bar shows:
- Current odds for each outcome
- Visual representation of market sentiment
- Updates in real-time with trading

### Volume

Volume shows total USDC traded:
- Includes all buys and sells
- Updates after each trade
- One of the key market health metrics

### End Date

Market deadline:
- Shows countdown for active markets
- Becomes "Ended" after deadline passes
- Then enters grace period for resolution

## Categories Explained

### Category Badges

Each market has a category badge:
- Easy identification
- Quick filtering
- Color-coded

## Finding Markets

### Strategies

1. **Browse All** - Start with newest
2. **Filter by Category** - Find your interest
3. **Search** - Find specific events
4. **Sort by Volume** - Find popular markets

### Popular Categories

Most active categories typically:
- Crypto (price predictions)
- Sports (game outcomes)
- Politics (election results)

## Market Detail

Click any market card to:
- View full description
- Read market rules
- Place trades
- See history

## Mobile Experience

Mobile view shows:
- Stacked card layout
- Horizontal scroll for categories
- Simplified sorting options

## Troubleshooting

### Markets not loading
- Check internet connection
- Refresh the page
- Verify wallet connection

### Filters not working
- Clear filter and reapply
- Check if market exists in category

### Search not finding
- Try different keywords
- Check spelling
- Markets may not exist yet
